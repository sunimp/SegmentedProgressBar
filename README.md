## Requirements

* Xcode 15.4+
* Swift 5.10+
* iOS 13.0+

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/sunimp/SegmentedProgressBar.git", .upToNextMajor(from: "1.0.0"))
]
```
